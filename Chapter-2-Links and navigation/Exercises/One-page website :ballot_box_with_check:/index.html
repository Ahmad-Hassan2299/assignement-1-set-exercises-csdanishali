Overview

8 Ball Pool is an online and mobile-based billiard-themed pool simulation sports game maintained and operated by Miniclip, a games company based in Switzerland, Portugal, Italy, and England. The game allows players to compete against each other in real-time, offering a realistic pool experience with various game modes and tournaments.

Key Information


Developer: Miniclip


Publisher: Miniclip


Platforms: iOS, Android, Web browser


Release Date: October 2010
